## ğŸ“¦ Installation & Setup

### Prerequisites
- **Node.js** (v18 or higher)
- **npm** or **yarn**

### Quick Start

```bash
# Clone the repository
git clone https://github.com/bondanbanuaji/modern-portfolio-starter-code.git

# Navigate to project directory
cd modern-portfolio-starter-code

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:5173 in your browser
```

### Build for Production

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

### Linting & Code Quality

```bash
# Run ESLint to check for code issues
npm run lint

# Run ESLint with auto-fix
npm run lint -- --fix
```

## ğŸ¨ Customization Guide

### Personal Information
Update `src/constants/data.jsx` to customize:
- Personal details (name, bio, contact info)
- Project portfolio
- Work experience
- Skills and technologies
- Social media links
- Achievements and metrics

### Styling
- Modify `src/index.css` for global styles
- Update Tailwind configuration in `tailwind.config.js`
- Customize colors and animations in `src/App.jsx`

### Animations
- Adjust animation properties in the component files
- Modify Framer Motion variants in `App.jsx`
- Customize gradient blob animations in `data.jsx`
